# Lazy Emacs Terminal


<a id="org0c661d7"></a>

## Install

Download the package

    git clone https://github.com/lizqwerscott/lazy-emacs-terminal.git .emacs.d

Install submodule

    cd .emacs.d
    git submodule init
    git submodule update
